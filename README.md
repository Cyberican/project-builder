# Project Builder

## Prerequisites:

* Maven 3.X.X or higher

``` sh

Project Builder
Execute a command	  [ exec ]
Flush Selected App	[ flush ]
View Help Menu		  [ help ]

COMMAND:
Build Application: build
Flush Application: flush
USAGE:
./project_builder.sh	--exec=COMMAND
./project_builder.sh	--exec=COMMAND --app=APP_NAME

```
